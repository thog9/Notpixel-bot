# Not Pixel BOT
BOT Click: [NotPixel Bot](t.me/notpixel/app?startapp=f921415493) 

## Features:

* Khai thác Pixel
* Vẽ Pixel
* Hiển thị số dư người dùng
* Chống phát hiện

## Module:

- Python 3.7 trở lên
- `pip` (trình cài đặt gói Python)

## Installation

2. **Install Module:**
   - Mở cmd hoặc Shell, sau đó chạy lệnh:
    ```sh
    pip install -r requirements.txt
    ```
3. **Chạy:**
- Mở cmd hoặc Shell, sau đó chạy lệnh:
    ```sh
    python bot.py
    ```
